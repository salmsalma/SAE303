À PROPOS DE CINEMMI

À propos de l'application

Bienvenue sur CineMMI, votre application interactive pour retrouver facilement tous les cinémas autour de vous ! Grâce à une carte dynamique, vous pouvez :

  • Visualiser tous les cinémas présents sur la carte
  • Rechercher un cinéma spécifique par nom ou localisation
  • Consulter rapidement les informations essentielles sur chaque cinéma

Technologies utilisées

Cette application a été développée en JavaScript, en utilisant les données de cinémas au format JSON (source : Data Île-de-France), et intégrée avec la bibliothèque Leaflet pour créer une carte interactive et intuitive. Les données sont chargées via l'API fetch, et un indicateur informe l'utilisateur pendant le chargement pour une expérience fluide.

L'objectif de ce projet est d'apprendre à manipuler des données interactives tout en respectant :

  • Les bonnes pratiques de git pour la gestion de version
  • Les règles d'accessibilité (contrastes, taille de texte, navigation simple)
